# 智慧城市课程结课论文代码复现

## 项目简介

本仓库包含**智慧城市课程**结课论文的配套代码复现。

论文题目：**特征驱动的手术室鲁棒调度：模型构建与数值复现**

## 项目说明

这是一个学术研究项目，旨在通过代码实现来复现论文中提出的特征驱动手术室鲁棒调度模型。本项目是智慧城市课程的结课作业，展示了如何将理论模型转化为可执行的代码实现。

## 研究内容

- 手术室调度优化
- 鲁棒性建模
- 特征驱动的调度方法
- 数值实验与结果验证

## 仓库文件说明

### 主要实验代码
- **`or_plus.py`** - **论文最终实验代码**，包含完整的手术室鲁棒调度模型实现和数值实验

### 论文理解阶段的开发文件
- `test_math_1.py` - 论文理解阶段的半成品代码（版本1）
- `test_math_2.py` - 论文理解阶段的半成品代码（版本2）
- `test_math_3.py` - 论文理解阶段的半成品代码（版本3）

### 改进过程中的废弃版本
以下文件在改进过程中因数学理解有误而被放弃：
- `or_scheduling_experiments.py` - 带有错误数学理解的废弃版本
- `or_scheduling_plus.py` - 带有错误数学理解的废弃版本
- `or_scheduling_simple.py` - 带有错误数学理解的废弃版本

## 使用说明

### 运行论文实验代码

